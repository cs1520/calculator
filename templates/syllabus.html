Programming Languages for Web Applications

Course Description

This course will focus on Web programming tools and technologies, both from the server side and the client side point of view. Server side technologies include Python and HTTP protocols. Client side technologies include the JavaScript scripting language (including JavaScript libraries), HTML, CSS, and AJAX protocols.

Prerequisites: CS445
